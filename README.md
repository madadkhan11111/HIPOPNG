# HipoPNG.com 🦛

**The World's Most Powerful Free Image Compressor.**

HipoPNG.com is a completely free, browser-based image optimization tool. It uses smart lossy compression techniques to reduce the file size of your WEBP, JPEG, and PNG files intelligently.

## Features
- **Smart Compression**: High-quality optimization for WebP, PNG, and JPEG.
- **Auto-Conversion**: Automatically convert images to AVIF, JXL, WEBP, JPEG, or PNG.
- **100% Private**: All processing happens locally in your browser. Your images never leave your computer.
- **Unlimited Usage**: No file size limits, no daily limits, completely free.
- **Cloud Integration**: Save directly to Dropbox or Google Drive.
- **User Friendly**: Simple drag-and-drop interface with a cute Hippo mascot.

## Technologies Used
- HTML5 / CSS3 (Modern Flexbox/Grid)
- JavaScript (Vanilla)
- [browser-image-compression](https://github.com/Donaldcwl/browser-image-compression)
- [JSZip](https://stuk.github.io/jszip/)
- [FileSaver.js](https://github.com/eligrey/FileSaver.js/)

## How to Run Locally
1. Clone this repository.
2. Open `index.html` in any modern web browser.
3. (Optional) Run a local server:
   ```bash
   python -m http.server 3000
   ```
